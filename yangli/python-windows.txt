在windows下用python，有以下几个选择

1. anaconda，
下载安装，在spyder或ipython notebook里使用。
anaconda现在安装容易出错。spyder好用，但调试不太方便。

2. 直接装pycharm，打开，菜单file, settings, Project, project interpreter, 可以添加安装需要的库。
pycharm调试方便。

2. 在viusal studio 里使用python
下载地址 https://github.com/Microsoft/PTVS
网站上有以下几个东西
Python Tools for VS 2015
Python Tools for VS 2013
Sample Pack
Machine Learning Pack

下载Python Tools for VS 2013 和 Machine Learning Pack，分别双击安装，安装的时候最好把vs关掉。
PTVS的好处有很多，调试方便，而且完全兼容了vs的使用习惯。
